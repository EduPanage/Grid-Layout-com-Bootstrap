# Grid Layout Bootstrap

Projeto desenvolvido com o objetivo de praticar e consolidar conhecimentos em **Bootstrap**, adquiridos durante uma especialização. A aplicação consiste em uma página web simples utilizando o sistema de **Grid Layout do Bootstrap**, explorando conceitos de responsividade, organização de conteúdo e estruturação de layouts modernos.

## 📖 Sobre o Projeto

Este projeto foi criado como exercício prático para aplicar conceitos fundamentais do Bootstrap, incluindo:

- Sistema de Grid (Rows e Columns)
- Estruturação semântica da página
- Organização de conteúdo em seções
- Responsividade para diferentes tamanhos de tela
- Utilização de componentes visuais e estilização complementar com CSS

A página possui uma estrutura composta por:

- **Cabeçalho (Header)**
- **Barra lateral (Aside)**
- **Conteúdo principal (Main)**
- **Rodapé (Footer)**

Além disso, foram utilizados textos ilustrativos (*Lorem Ipsum*) e imagens para simular um cenário real de desenvolvimento.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Bootstrap 5
